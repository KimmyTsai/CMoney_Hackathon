# 🌳 AI 投資樹洞（AI Investment Companion）

> **CMoney × AWS AI Hackathon 2026** — 重新定義股票庫存管理
>
> 過去的「股票庫存」是記帳工具：輸入幾筆、看一眼損益、就不再回來。
> 我們讓 AI 讀懂你的真實持股，從「回答通用問題」進化到「回答**這個人現在最想問的問題**」。

**時間基準**：一切以 `2025/12/31` 為「今天」｜**資料**：CMoney 官方 300 檔示範籃子（11 個資料表）

---

## ✨ 核心功能

| 功能 | 說明 |
|---|---|
| 📥 **低門檻持股導入** | 手動輸入（代號/名稱即時搜尋、成本留 0 自動以年均價估算）＋ **AI 視覺辨識截圖**（Bedrock 多模態直接讀看盤 App 截圖與對帳單，逐筆保留成本/股數/買進日期，支援只有股票名稱的表格） |
| ⚡ **即時洞察卡** | 加入第一檔的瞬間就給回饋：帳面 vs 含息總報酬、成本殖利率、成本買點分位、連續配息年數、股利遞增/遞減趨勢 |
| 🫂 **匿名同儕對照** | 重虧持股顯示「2025 年○○同學會有 N 萬則討論、全年落後大盤 X%——套牢的不只你」，用社群數據接住情緒 |
| 🌳 **組合層 AI 診斷** | 整個組合**一份**報告（一次 Bedrock 呼叫）：投資風格判讀開場、帳面 vs 含息真相、產業/市值配置、集中度、越攤越平偵測、同溫層提醒、槓桿型 ETF 性質辨識 |
| 🧩 **產業配置視覺化** | 依市值的產業比例條，與 AI 報告、人格卡數值完全一致 |
| 🛡️ **持股防護罩** | 回訪機制的實體：四類警示引擎（法人連 5 日買賣超、創年高/低、社群聲量爆量與情緒轉空、除息事件），以 **2025 全年真實資料回放**證明通知價值，法人/動能/除息警示附日期區間限定的新聞溯源連結 |
| 🎴 **投資人格卡** | 匿名化的社交傳播：AI 依成本分位/殖利率結構/持有天數判定人格，9:16 抽卡風 SVG 可下載分享，代號打碼模式以產業聚合（與產業配置數值一致） |

### 命題「AI 需要做到的六件事」對應

| 能力 | 實作 |
|---|---|
| 投資風格 | AI 報告第一句話的風格判讀（成本分位 × 殖利率結構 × 持有天數 × 商品性質） |
| 資產配置 | 產業配置比例圖＋市值分佈（ETF/大型/中小型） |
| 集中風險 | 逐檔權重、最大單一權重、>50% 系統警示 |
| 投資習慣 | 成本買點分位（含拆分偵測）＋逐批次持有天數（多批次買進模型） |
| 常見錯誤 | 越攤越平偵測（成本分位 vs 現價分位 × 重虧 × 持有天數）；賺小賠大列於 Roadmap（需逐筆交易 API） |
| 決策焦慮 | 近 20 日法人買賣超 ＋ 同學會 30 日多空比/聲量變化雙訊號 |

## 🏗️ 系統架構

```
┌─ Streamlit Web App (web-app/) ─────────────────────────────┐
│  app.py            三步驟流程 / 防護罩 / 人格卡 / 深淺色主題   │
│  data_processor.py 資料引擎（含息報酬、分位、社群脈動、警示）    │
│  llm_service.py    AI 引擎（組合診斷 prompt、視覺 OCR、法遵）  │
│  persona_service.py 人格判定與 SVG 卡片生成                  │
│  ocr_service.py    EasyOCR 本地備援                         │
└──────────────┬─────────────────────────┬───────────────────┘
               │ 主引擎                    │ 隱私模式 / 離線備援
        Amazon Bedrock              Ollama (本地)
        Claude（文字診斷＋視覺辨識）   qwen2.5:7b-instruct
               │
        CMoney 資料包（11 表：行情/法人/報酬/動能/股利/社群…）
```

**設計原則**
- **模型抄錄、程式算數**：所有數值計算（加權成本、佔比、分位、警示規則）在確定性程式層完成，LLM 只負責理解與表達
- **輸出全數 grounding**：未來除息日等事實只允許引用資料集欄位，prompt 明令禁止推測未提供的資訊
- **法遵**：全程禁止「買進/賣出/加碼/減碼」等直接投資建議，一律使用「值得想想／可以留意」觀察框架
- **資料防呆**：0050 年中 1 拆 4 的拆分偵測（分位/年均價/警示引擎一致處理）、負數配息年數語意轉換、槓桿型 ETF 標記

## 🚀 快速啟動

```bash
cd web-app
pip install -r requirements.txt

# ── AWS Bedrock（主引擎）──
aws configure                        # 或以環境變數提供臨時憑證
export BEDROCK_REGION=us-east-1      # 依主辦方指定 region
export BEDROCK_MODEL_ID=$(aws bedrock list-inference-profiles --region us-east-1 \
  --query "inferenceProfileSummaries[?contains(inferenceProfileId,'claude')].inferenceProfileId" \
  --output text | tr '\t' '\n' | head -1)   # 或手動填入探測可用的模型 ID

# ── 資料路徑 ──
export CMONEY_DATA_DIR=../cmoney-aws-summit-hackathon/Delivery_Hackathon_DataPackage_20260624

streamlit run app.py
```

Windows PowerShell 對應寫法：`$Env:BEDROCK_REGION="us-east-1"`（其餘同理）。

**Ollama 本地隱私模式（選用）**：安裝 [Ollama](https://ollama.com) 後 `ollama pull qwen2.5:7b-instruct`，側邊欄即可切換——持股資料與 AI 運算全程不離開你的電腦。Bedrock 不可用時亦自動降級至本地引擎。

## 🎬 Demo 動線（3 分鐘）

1. **導入**：按「一鍵載入示範組合」（00919×10000@23.5、兆豐金×3000@42、台泥×2000@38），或上傳對帳單截圖體驗 AI 視覺辨識
2. **即時回饋**：持股卡當下顯示「00919 帳面 -4.7% 但含息 +6.0%」＋台泥觸發同儕對照
3. **健診**：組合存摺（帳面 vs 含息 vs 0050）、系統警示（集中度 57%／00919 同溫層多空比 126×／台泥攤平提醒）
4. **AI 報告**：風格判讀開場的組合診斷＋產業配置圖＋真實除息日回訪鉤子
5. **翻牌**：抽出投資人格卡，打碼分享
6. **訂閱防護罩**：氣球金幣慶祝 → 警示中心展示 2025 全年 127 則真實通知回放
